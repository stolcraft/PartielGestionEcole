Bienvenue sur GestionEcole

==========================================
Le projet comporte les modules suivants :

==========================================

1\  Le projet possède une documentation
disponible dans le dossier documentation

2\ Le projet est réalisé sous maven avec
les différents modules sont écrit dans
le pom.xml

3\ Les test des méthodes sur la couche
service ont été éfféctué avec mockito 
et le système de Mock

4\ Toutes les classes sont disponibles
dans le jar "gestionEcole" avec les tests
de la couche service

5\ Les deux diagrammes UML sont disponible
dans le dossier UML

6\Importer ecolegestion.sql dans la BDD ecolegestion

7\Installer TomcatServer 9.0 1-Deplacer le 
fichier GestionEcole3.war dans le dossier webapp de 
Tomcat 2-Executer startup.bat situé dans le dossier 
bin de tomcat 3-Dans l'url du navigateur internet et ensuite allez sur le localhost
