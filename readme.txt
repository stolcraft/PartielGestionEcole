Bienvenue sur GestionEcole

==========================================
Le projet comporte les modules suivants :

==========================================

1\  Le projet possède une documentation
disponible dans le dossier documentation

2\ Le projet ne possède pas d'interface
Web car je n'ai pas eu le temps

3\ Le projet est réalisé sous maven avec
les différents modules sont écrit dans
le pom.xml

4\ Les test des méthodes sur la couche
service ont été éfféctué avec mockito 
et le système de Mock

5\ Toutes les classes sont disponibles
dans le jar "gestionEcole" avec les tests

6\ Les deux diagrammes UML sont disponible
dans le dossier UML

7\ Pour la création de la base de donnée
il faut créer une nouvelle BDD "gestionecole
et ensuite JPA va créer tout seul la BDD
